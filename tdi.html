demo site →

type: website
status: in development
tools/stack: typescript, react.js, next.js


why

Tenacious Diagnostics is a pre-seed medical device startup, aiming to treat dysphagia through an implantable medical device. The 2 co-founders have been working to prototype and patent, while I cover operations/GTM/Bizdev. While we are still early stages, I sought to mock a demo page with Claude Code. Note that most of the content on this site is a placeholder at the moment, as we are still conducting product research.


what

landing page features:
  • call to action: solution and evidence
  • audience tabs: targeted info specialized to end users, providers, and payers
  • investor page: separate investor-specific landing page

how

This app was purely vibecoded through prompting Claude code in my terminal.

reflections

This was my first time using Clade Code, and it was incredible to see how the coding process has fundamentally changed. The time to develop this initial demo site took maybe an hour. For this site, there are definitely future iterations that I will be creating, but it was a really great learning experience with Claude Code, especially its Plan mode.
