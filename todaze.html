todaze

tldr: social media for to-do lists



type: web app
status:deployed
tools/stack: html, css, javascript, firebase


overview

Todaze is a social productivity app that allows users to share to-do lists with friends.

The idea emerged from wanting accountability without the pressure of formal productivity systems.

Instead of optimizing tasks, the app focuses on visibility and lightweight social motivation.


features

  • Create and manage personal tasks
  • View friends' task lists
  • Real-time updates
  • Minimalist interface

reflections

This project taught me a great deal about designing social experiences and balancing simplicity against functionality.

Future versions could include richer social interactions, recurring tasks, and public accountability groups.
